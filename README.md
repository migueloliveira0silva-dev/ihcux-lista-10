# ihcux-lista-10

Miguel Oliveira Silva

Minha ideia foi fazer um app para um hospital para informar e marcar consultas

### 1. Tela de Login/Acesso
* **Função:** É a porta de entrada para o sistema.
* **Elementos:** Identificação do hospital, campos para **CPF** e **Senha**, e botões destacados para **Entrar** ou **Cadastrar**.
* **Ação:** Ao clicar em "Entrar", o usuário é levado para a tela principal de seleção de serviços.

### 2. Tela de Seleção de Unidade/Serviço
* **Função:** Apresenta a unidade do hospital com informações rápidas.
* **Elementos:** Foto da fachada, avaliação (4.0 estrelas), distância (50m) e status de funcionamento (Aberto 24h). 
* **Ações:** Possui três botões principais:
    * Um seletor para "Unidade Única".
    * **Agendar Consultas:** Direciona para a lista de espera/atendimento.
    * **Informações do Estabelecimento:** Direciona para detalhes institucionais.

### 3. Tela de Detalhes do Estabelecimento
* **Função:** Oferece uma visão mais institucional e visual do hospital.
* **Elementos:** Galeria de fotos da estrutura interna e externa, logotipos e informações de horários detalhados. É a tela para o usuário conhecer a infraestrutura antes de prosseguir.

### 4. Tela de Lista de Espera (Painel de Atendimento)
* **Função:** Monitorar em tempo real o tempo para ser atendido.
* **Elementos de Destaque:**
    * **Tempo Médio de Espera:** Exibe o cálculo que fizemos anteriormente (**28 min e 20 seg**).
    * **Status Individual:** Mostra os médicos (ou pacientes) com seus respectivos tempos: Andre Freitas (**5 min**), Claudia Marques (**20 min**) e Luiz Carlos (**1 HORA**).
* **Utilidade:** Garante transparência para o usuário sobre quanto tempo ele terá que aguardar no local.
